Veb tərtibatçısı, müştəri-server modelindən istifadə edərək World Wide Web proqramlarını inkişaf etdirən proqramçıdır. Tətbiqlər adətən müştəridə HTML, CSS və JavaScript-dən və serverdə hər hansı ümumi təyinatlı proqramlaşdırma dilindən istifadə edir. HTTP müştəri və server arasında əlaqə üçün istifadə olunur. Veb tərtibatçısı müştəri tərəfi proqramlar (Front-end web inkişafı), server tərəfi proqramlar (back-end inkişafı) və ya hər ikisində (tam stack inkişafı) ixtisaslaşa bilər.

HTML

HTML proqramlaşdırma dili kimi müəyyən edilə bilməz. Çünki öz-özünə işləyən proqram HTML kodları ilə yazıla bilməz. Bununla belə, bu dili şərh edə bilən proqramlar vasitəsilə işləyə bilən proqramlar yazıla bilər. Məhz buna görə də onu proqramlaşdırma dili adlandırmaq olmaz. Əslində mətn, şəkillər, videolar və onları ehtiva edən səhifələr kimi müxtəlif məlumatları bir-birinə bağlamaq, həmçinin veb-brauzer proqramı ilə sözügedən səhifələrin düzgün göstərilməsi üçün lazımi qaydaları müəyyən etməkdir. Veb brauzerlər HTML kodunu oxuyur, şərh edir və vizual formaya çevirir, beləliklə, eyni HTML kodunun müxtəlif brauzerlərdə fərqli nəticələr verməsi mümkündür. CSS və JavaScript ilə istifadə edildikdə, HTML vasitəsilə vizual və dinamik veb saytlar yaradıla bilər. HTML kiçik və böyük işarələri arasında olan HTML teqlərindən istifadə etməklə yazılır (məsələn, ). Onlar əsasən işarələnmiş mətnin əvvəlini və sonunu göstərmək üçün cüt-cüt istifadə olunur.

CSS

Cascading Style Sheets (Cascading Style Sheets, həmçinin CSS kimi tanınır) HTML ilə yanaşı mətn və formatlaşdırma sahəsində əlavə imkanlar təklif edən işarələmə dilidir. Veb səhifələr üçün ümumi şablonlar hazırlamaq imkanı verməklə yanaşı, həm də hərflərin üslubunu, yəni rəng, şrift, ölçü kimi xüsusiyyətlərini müstəqil olaraq dəyişdirmək üçün istifadə edilə bilər. Bu texnikanın ən mühüm xüsusiyyəti istifadədə çevikliyidir. Veb səhifə daxilində bir neçə uyğun rəng və bir neçə şrift istifadə olunur və onları hər bir səhifədə ayrı-ayrılıqda yenidən müəyyən etmək əvəzinə, CSS-in köməyi ilə bir dəfə müəyyən edilə və bütün Web səhifələrində ümumi istifadə oluna bilər. Beləliklə, yaddaşdakı səhifələrin ölçüsü xeyli kiçik olur və onları yeniləmək daha asan olur.

JAVASCRIPT

Qrafik istifadəçi interfeysi olan ilk məşhur veb-brauzer, Mosaic, 1993-cü ildə buraxıldı. Qeyri-texniki insanlar üçün əlçatan olan bu, yeni yaranan Ümumdünya Şəbəkəsinin sürətli böyüməsində mühüm rol oynadı.[12] Mosaic-in aparıcı tərtibatçıları daha sonra 1994-cü ildə daha cilalanmış brauzer Netscape Navigator-u buraxan Netscape korporasiyasını qurdular. Bu, tez bir zamanda ən çox istifadə edilən oldu. Vebin bu formalaşma illərində veb-səhifələr yalnız statik ola bilərdi, səhifə brauzerə yükləndikdən sonra dinamik davranış qabiliyyətindən məhrum idi. Çiçəklənən veb inkişaf səhnəsində bu məhdudiyyəti aradan qaldırmaq arzusu var idi, buna görə də 1995-ci ildə Netscape Navigator-a skript dili əlavə etmək qərarına gəldi. Onlar buna nail olmaq üçün iki yol izlədilər: Java proqramlaşdırma dilini yerləşdirmək üçün Sun Microsystems ilə əməkdaşlıq etmək, eyni zamanda Sxem dilini yerləşdirmək üçün Brendan Eyxi işə götürmək..

BOOTSTRAP

Bootstrap HTML, CSS və JS kitabxanasıdır və əsas diqqəti informativ veb səhifələrin inkişafının sadələşdirilməsinə yönəldir (veb proqramlarından fərqli olaraq). Onu veb layihəyə əlavə etməyin əsas məqsədi Bootstrap-ın rəng, ölçü, şrift və tərtibat seçimlərini həmin layihəyə tətbiq etməkdir. Beləliklə, əsas amil cavabdeh olan tərtibatçıların bu seçimləri öz zövqlərinə uyğun tapıb tapmamasıdır. Layihəyə əlavə edildikdən sonra Bootstrap bütün HTML elementləri üçün əsas üslub təriflərini təqdim edir. Nəticə veb brauzerlərdə proses, cədvəllər və forma elementləri üçün vahid görünüşdür. Bundan əlavə, tərtibatçılar məzmunlarının görünüşünü daha da fərdiləşdirmək üçün Bootstrap-da müəyyən edilmiş CSS siniflərindən istifadə edə bilərlər. Məsələn, Bootstrap açıq və tünd rəngli cədvəllər, səhifə başlıqları, daha nəzərə çarpan sitatlar və vurğulanmış mətn üçün təminat verib. .

React (həmçinin React.js və ya ReactJS kimi tanınır) komponentlərə əsaslanan istifadəçi interfeysləri yaratmaq üçün pulsuz və açıq mənbəli JavaScript kitabxanasıdır. O, Meta (əvvəllər Facebook) və fərdi tərtibatçılar və şirkətlər icması tərəfindən qorunur. React-dən Next.js kimi çərçivələrlə tək səhifəli, mobil və ya server tərəfindən göstərilən proqramlar hazırlamaq üçün istifadə oluna bilər. React yalnız istifadəçi interfeysi və DOM-a komponentlərin göstərilməsi ilə məşğul olduğundan, React proqramları marşrutlaşdırma və digər müştəri funksiyaları üçün çox vaxt kitabxanalara etibar edir.

2006-cı ildə Con Reziq tərəfindən istifadəyə buraxılmışdır. Hal-hazırda Dave Methvin rəhbərlik etdiyi komanda tərəfindən inkişaf etdirilir. jQuery JavaScript kitabxanaları arasında ən çox istifadə olunan kitabxanadır. 10,000 ən çox ziyarət edilən saytın 55%-də jQuery istifadə edilib. CSS, HTML strukturunu vizuallaşdırır, jQuery HTML strukturunu hərəkətləndirir. Məsələn, düymənin basılması ilə emalçı birbaşa göstərişin yerinə idarəni JQuery-ə ötürülür, burada da, düymələr müəyyən edilir və sonra düyməyə basıldıqda emalçı onu təyin olunan funksiyaya uyğunlaşdırır. Davranışın və strukturun belə bölməsi Unobtrusive JavaScript adlanır. JQuery kitabxanası, tapşırıqların maksimum geniş mühiti üçün faydalı funksionallığı mövcuddur. Bununla belə, jQuery-dəki funksiyaların uyğunlaşdırma məsələsi kitabxana istehsalçıları tərəfindən təyin edilmir, çünki, bu mənbə kodunun böyüklüyünə və mürəkkəbliyinə səbəb olur. Buna görə kitabxananın və plaqinlərin yığcam, universal nüvə arxitekturası yaradılmışdır. Bu resurs məhz funksional-JavaScript yığmağa imkan verir.

PHP (ing. PHP: Hypertext Preprocessor) dinamik veb səhifələr yaratmaq üçün nəzərdə tutulmuş bir skriptləşdirmə dilidir.PHP Əsasən veb həllərin yaradılmasında istifadə edilən populyar server proqramlaşdırma dilidir. Məsələn, formadakı məlumatları qəbul edərək emal edən skript yaratmaq, saytda axtarış sistemi reallaşdırmaq və ya qeydiyyat sistemi yaratmaq mümkündür. 1994-cü ildə Rasmus Lerdorf CGI alətlərini yazmağa başladı və daha sonra 1995-ci ildə onları mükəmməlləşdirərək PHP dilini ortaya çıxartdı. Əksər hallarda PHP, Linux əməliyyat sistemi, MySQL məlumatlar bazası və Apache veb serveri qısa olaraq LAMP kimi (Linux, Apache, MySQL, PHP) yazılır. PHP dili 1995-ci ildə yaradılmışdır. İlk əvvəllər onun adı Personal Home Page sözlərinin baş hərflərinin birləşməsindən əmələ gəlmişdir. Daha sonralar daha ciddi ad fikirləşmək məqsədi ilə Hypertext Preprocessor adı verildi. Lakin HPP bir o qədər yaxşı səslənmədiyindən PHP adı qaldı. 2004-cü ildən PHP obyekt yönümlü dil kimi istifadə olundu. Məhz bu ildə Zend şirkəti PHP üçün yeni obyekt modeli tətbiq etməyə başladı. PHP5 versiyasında artıq obyektlər üzvləri(metodlar, xassələr) private, protected, public, static, final kimi ola bilər, interfeyslərdən istifadə oluna bilər. 2009-cu ildə Zend şirkəti PHP üçün Zend Server platformasını yaratdı. Zend Serverdə PHP kodlar kompilyasiya olunub bayt-kodlar şəklində qalır və bu da öz növbəsində kodun daha sürətli işləməsinə səbəb olur.
